# DS18B20 Temperature Exporter

For exposing a DS18B20 sensor as a prometheus metric. Developed on Raspberry Pi Zero W.

# Usage

```
sudo apt-get install --fix-missing -y python3-pip
sudo pip3 install -r requirements.txt
./temp-exporter.py
```
